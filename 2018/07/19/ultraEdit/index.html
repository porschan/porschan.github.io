    UltraEdit常用快捷键

    F12 ：另存为

    F7 ：插入当前日期时间

    Ctrl + N ：创建一个新文件

    Ctrl + O ：打开文件

    Ctrl + S ：保存活动文件

    Ctrl + E ：删除当前行

    Ctrl + G ：跳到指定行

    Ctrl + U ：打开文件树

    Ctrl + F4 ：关闭文件

    Ctrl + F5 ：转为小写

    Ctrl + D ：删除当前行

    ALT + F5 ：转为大写

    Alt + F3 or Ctrl+F ：查找
